# linkedin-job-analysis
This project explores LinkedIn job listings to understand current hiring trends and identify the most in-demand skills in the job market.

I worked with real-world job data and used Python to clean, analyze, and extract insights that can help job seekers understand what employers are looking for.

What I did in this project:

- Cleaned and prepared raw job listing data
- Analyzed job titles, locations, and required skills
- Identified patterns in hiring trends
- Explored which skills appear most frequently

Tools used:

Python
Pandas
Jupyter Notebook

What you can find here:

The notebook contains the full analysis, including data cleaning, exploration, and insights.
